Курс по WEB-разработке Материалы станут доступны в субботу 07.11.2015 г. Курс подразумевает, что студенты уже освоили базовый python. В рамках курса даются азы html/css, достаточные для формирования html страниц браузеру, но не достаточные для профессиональной верстки. Глубина освоения django будет достаточной для самостоятельного написания web-проектов, которые содержат редактируемые страницы, структуру специфических данных, отображение их на фронтенде и создание-редактирование этих данных. Все разделы django будут освещены.
