Description PyBursa

Этот курс знакомит студентов с языком Python. После завершения этого класса, студент сможет писать не тривиальные программы на Python, касающиеся широкого спектра предметных областей. Темы включают языковые компоненты, среда разработки IDLE, конструкции потока управления, строки ввода / вывода, коллекции, классы, модули и регулярные выражения.Курс дополнен многими лабораторными с использованием либо Linux или Windows.
